# Bootcamp-Challenge-2
Second challenge of the Fullstack Web Development Bootcamp

Objective:
Create a simple browser-based game using JavaScript (DOM)

The game is Rock Paper Scissor
1. From the home page (mySite.html), user click on the Trial Button.
2. User redirected to the trial page (trial.html)
3. User click one of three buttons (rock/paper/scissor) on the left/player 1 side
4. Computer/script will choose a random selection of rock/paper/scissor
5. Both player and computer selected buttons will be highlighted by background color change
6. The match result is shown (Player 1 Win/COM Win/Draw)

Player 1, COM move, and match result is shown on the browser console

7. To restart the game, user click the restart button at the bottom of the page
8. To go back to the home page, user click on the "<" sign at the top left corner of the page
